# CSE4062S26_Grp2 - Diabetes Data Science Project

## Project Overview
This project involves a complete data science pipeline for the **Diabetes 130-US Hospitals for Years 1999–2008** dataset. 

### Objectives
1. **Data Understanding and Preprocessing**: Identifying feature types, missing values, and preparing the final dataset.
2. **Descriptive Analytics**:
    - **Clustering**: k-means, hierarchical clustering, DBSCAN.
    - **Association Rule Mining**: Apriori.
3. **Predictive Analytics**:
    - **Classification**: Predicting early readmission or other targets.

### Methodology
We use the **Autoresearch** framework (inspired by Andrej Karpathy) to autonomously experiment with feature selection and model optimization.
