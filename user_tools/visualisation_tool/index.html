    Drop Criteria — Near-Zero Variance (NZV) Analysis

    Applied to entire dataset — Kuhn & Johnson (2013) & Pedregosa et al. (2011)

    A — Frequency Ratio

    FR = count(most common) / count(2nd most common)

    Drop if FR > 20

    B — Uniqueness %

    UP = (n_unique / n_total) × 100

    Drop if UP < 10%

    C — Variance Threshold

    σ² = p × (1−p)  |  t = 0.0475

    Drop if σ² < 0.0475 (categorical only)

    Decision Rule

    DROP if (A AND B) OR C

    Numeric columns: only criterion A & B apply

    Column Type FR UP (%) σ² NZV (A&B)? Var<t (C)? Decision
